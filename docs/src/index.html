주요 기능

유연한 조건 설정으로 원하는 시간에 알림을 받으세요

조건 기반 알림 설정

단순 타이머가 아닌, 원하는 조건을 직접 설정하여 정확한 시점에 알림을 받을 수 있습니다.

다양한 시간 조건

시간 범위, 반복 간격, 특정 시각 등 다양한 조건 타입으로 세밀하게 알림 규칙을 구성하세요.

AND/OR 복합 조건

논리 연산자로 여러 조건을 조합하여 복잡한 알림 규칙도 간단하게 만들 수 있습니다.

백그라운드 알림

네트워크 연결 없이 로컬에서 동작합니다. 가볍고 빠르게, 원하는 순간에 알림을 받으세요.
